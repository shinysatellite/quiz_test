## Merch Informer Project.

- Laravel v8.0
- Vue v2.6.14.
- Bootstrap v4.64
- Theme [OneUI Vue Edition](https://pixelcave.com/products/oneui-vue-edition)
## Install

- install vendors 

    `composer update`

- install  node_modules
    `npm install`

- install enviroments

    `cp .env.example .env`

    `php artisan key:generate`
    `php artisan migrate`
## Run
- `php artisan serve`
- `npm run watch`
